Core Pattern

The Neyman-Pearson move is to reject where the likelihood under the alternative is large compared with the likelihood under the null.

\[\Lambda(x)=\frac{L_1(x)}{L_0(x)},\qquad \text{reject }H_0\text{ when }\Lambda(x)>c,\qquad P_{H_0}(\text{reject})=\alpha.\]

Setup Checklist

  • State \(H_0\), \(H_1\), and the statistic.
  • Write the likelihood ratio before simplifying inequalities.
  • Choose the rejection tail from the ratio direction.
  • Calibrate size using the null distribution.
  • Compute power using the alternative distribution.
  • For UMP, look for monotone likelihood ratio.

Problems 1-5 - Concept Builders
1. Concept: simple Bernoulli likelihood ratio
ConceptLikelihood ratio

One observation \(X\sim\operatorname{Bernoulli}(p)\). Compare \(H_0:p=1/3\) against \(H_1:p=2/3\). Write \(L_1/L_0\) for \(X=0\) and \(X=1\), and identify the rejection value.

Solution

For \(X=1\), \(L_1/L_0=(2/3)/(1/3)=2\). For \(X=0\), \(L_1/L_0=(1/3)/(2/3)=1/2\). The ratio is larger when \(X=1\), so the rejection value is \(X=1\).

2. Concept: binomial right-tail direction
ConceptMonotone

For \(X\sim\operatorname{Binomial}(n,p)\), explain why testing larger \(p\) against smaller \(p\) uses large values of \(X\).

Solution

For \(p_1>p_0\), the ratio \(L_1/L_0\) is proportional to \(\left(\frac{p_1(1-p_0)}{p_0(1-p_1)}\right)^X\). The base is greater than \(1\), so the ratio increases with \(X\). Large \(X\) supports the larger \(p\).

3. Concept: size for a right-tail test
ConceptSize

If \(X\sim\operatorname{Binomial}(10,p)\), a test rejects for \(X\ge8\). Write the size under \(H_0:p=1/2\).

Solution

The size is the null rejection probability: \(\sum_{x=8}^{10}\binom{10}{x}(1/2)^{10}\).

4. Concept: power under the alternative
ConceptPower

For the same rejection rule \(X\ge8\), write the power at \(p=3/4\).

Solution

The power is \(P_{p=3/4}(X\ge8)=\sum_{x=8}^{10}\binom{10}{x}(3/4)^x(1/4)^{10-x}\).

Problems 6-8 - Integration Problems
6. Integration: normal size calibration
IntegrationSize

\(X_1,\ldots,X_n\sim N(\mu,\sigma^2)\) with known \(\sigma\). For \(H_0:\mu=\mu_0\) against \(H_1:\mu>\mu_0\), write a size \(\alpha\) rejection rule using \(z_{1-\alpha}\).

Solution

Under \(H_0\), \(Z=\sqrt n(\bar X-\mu_0)/\sigma\sim N(0,1)\). Reject when \(Z>z_{1-\alpha}\), equivalently \(\bar X>\mu_0+\sigma z_{1-\alpha}/\sqrt n\).

7. Integration: power for the normal right-tail test
IntegrationPower

For the rule in Problem 6, express the power at \(\mu=\mu_1>\mu_0\).

Solution

The power is \(P_{\mu_1}(\bar X>\mu_0+\sigma z_{1-\alpha}/\sqrt n)\). Standardizing under \(\mu_1\), it equals \(1-\Phi(z_{1-\alpha}-\sqrt n(\mu_1-\mu_0)/\sigma)\).

8. Integration: boundary randomization note
IntegrationRandomization

A discrete test statistic rejects for \(T>c\) with null probability \(0.03\), while \(P_{0}(T\ge c)=0.08\). Explain how to get exact size \(0.05\).

Solution

Reject always when \(T>c\), and reject with probability \(\gamma\) when \(T=c\). The boundary mass is \(0.08-0.03=0.05\). Solve \(0.03+\gamma(0.05)=0.05\), giving \(\gamma=0.4\).
