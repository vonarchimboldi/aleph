Core Pattern

Write the likelihood with its parameter space and support before differentiating.

\[L(\theta)=\prod_{i=1}^n f_\theta(x_i),\qquad \ell(\theta)=\log L(\theta),\qquad \widehat\theta=\arg\max_{\theta\in\Theta}L(\theta).\]

Setup Checklist

  • Identify the model, parameter, and parameter space.
  • Keep support restrictions that involve the parameter.
  • Use the log only after writing the likelihood.
  • Separate derivative candidates from endpoint candidates.
  • Check whether the likelihood is monotone on its allowed set.
  • For estimator quality, compute bias before MSE.
Problems 1-5 - Concept Builders
1. Concept: Bernoulli likelihood setup
ConceptLikelihood

\(X_1,\ldots,X_n\) are iid Bernoulli\((p)\), and \(s=\sum X_i\). Write the likelihood and find the MLE of \(p\).

Solution

The likelihood is \(L(p)=p^s(1-p)^{n-s}\), \(0\le p\le1\). Maximizing the log gives \(\widehat p=s/n=\bar X\), with endpoints included when \(s=0\) or \(s=n\).

2. Concept: exponential rate log-likelihood
ConceptLog

\(X_i\) are iid Exponential\((\lambda)\) with density \(\lambda e^{-\lambda x}\), \(x>0\). Find the MLE of \(\lambda\).

Solution

\(\ell(\lambda)=n\log\lambda-\lambda\sum x_i\). The derivative gives \(n/\lambda-\sum x_i=0\), so \(\widehat\lambda=n/\sum x_i=1/\bar X\).

3. Concept: normal mean with known variance
ConceptParameter

\(X_i\sim N(\mu,\sigma^2)\) iid with known \(\sigma^2\). Find the MLE of \(\mu\).

Solution

The log-likelihood differs from a constant by \(-\frac{1}{2\sigma^2}\sum (x_i-\mu)^2\). Maximizing it means minimizing the sum of squared deviations, giving \(\widehat\mu=\bar X\).

4. Concept: uniform endpoint support
ConceptSupport

\(X_i\) are iid Uniform\((0,\theta)\). Write the likelihood with support and find the MLE.

Solution

The likelihood is \(L(\theta)=\theta^{-n}\mathbf{1}_{\theta\ge x_{(n)}}\). Since \(\theta^{-n}\) decreases as \(\theta\) grows, the MLE is \(\widehat\theta=X_{(n)}\).

5. Concept: bias of the sample mean
ConceptBias

If \(X_1,\ldots,X_n\) have mean \(\theta\), show that \(\bar X\) is unbiased for \(\theta\).

Solution

By linearity, \(E[\bar X]=(1/n)\sum E[X_i]=(1/n)n\theta=\theta\). The bias is zero.

Problems 6-8 - Integration Problems
6. Integration: Poisson MLE and variance estimate
IntegrationMLE

\(X_i\) are iid Poisson\((\lambda)\). Find the MLE of \(\lambda\) and give the plug-in estimate of \(\operatorname{Var}(X_1)\).

Solution

Ignoring constants not involving \(\lambda\), \(\ell(\lambda)=(\sum x_i)\log\lambda-n\lambda\). The MLE is \(\widehat\lambda=\bar X\). Since a Poisson variable has variance \(\lambda\), the plug-in variance estimate is \(\bar X\).

7. Integration: shifted uniform boundary
IntegrationBoundary

\(X_i\) are iid Uniform\((\theta,\theta+1)\). Find the set of \(\theta\) values for which the likelihood is positive, then describe the MLE set.

Solution

The support requires \(\theta\le x_i\le\theta+1\) for every \(i\), so \(\theta\le x_{(1)}\) and \(\theta\ge x_{(n)}-1\). On this interval the likelihood equals \(1\), so every \(\theta\in[x_{(n)}-1,x_{(1)}]\) is an MLE.

8. Integration: MSE from bias and variance
IntegrationMSE

An estimator \(T\) satisfies \(E[T]=\theta+a\) and \(\operatorname{Var}(T)=v\). Compute its MSE for estimating \(\theta\).

Solution

\(\operatorname{MSE}(T)=\operatorname{Var}(T)+\operatorname{Bias}(T)^2=v+a^2\).

Problems 9-10 - ISI-Style Challenge
9. ISI-style challenge: decreasing density support
ISI-styleSupport

\(X_i\) are iid with density \(f_\theta(x)=\theta x^{\theta-1}\) on \(00\). Find the MLE of \(\theta\).

Solution

The log-likelihood is \(\ell(\theta)=n\log\theta+(\theta-1)\sum\log x_i\). The derivative is \(n/\theta+\sum\log x_i\). Setting it to zero gives \(\widehat\theta=-n/\sum\log x_i\), which is positive because each \(\log x_i<0\).

10. ISI-style challenge: endpoint estimator bias
ISI-styleBias

For iid Uniform\((0,\theta)\), the MLE is \(X_{(n)}\). Find \(E[X_{(n)}]\) and an unbiased estimator based on \(X_{(n)}\).

Solution

\(X_{(n)}/\theta\) has beta distribution with parameters \(n,1\), so \(E[X_{(n)}]=n\theta/(n+1)\). Therefore \((n+1)X_{(n)}/n\) is unbiased for \(\theta\).

Answer Summary

  5. Bias \(0\)
  6. \(\bar X\), variance estimate \(\bar X\)
  7. All \(\theta\in[X_{(n)}-1,X_{(1)}]\)
  8. \(v+a^2\)
