Core Pattern

Turn the target count into a sum of indicators, then compute one event probability and multiply by the size of the index set.

\[X=\sum_{\alpha\in A} I_\alpha,\qquad E[X]=\sum_{\alpha\in A}P(I_\alpha=1),\qquad E[X(X-1)]=\sum_{\alpha\ne\beta}P(I_\alpha I_\beta=1).\]

Setup Checklist

  • Name the atomic object before naming the indicator.
  • Use symmetry only after the same probability applies to every atom.
  • For conditioning, compute \(E[X\mid Y]\) first.
  • For variance, classify identical, overlapping, and disjoint pairs.
  • For unordered pairs, decide whether \(X(X-1)\) needs ordered pairs.
  • Write one sentence explaining why independence is or is not needed.
Problems 1-5 - Concept Builders
1. Concept: prime-numbered successful trials
ConceptLinearity

Ten independent trials succeed with probability \(p\). Let \(X\) count successes in positions \(2,3,5,7\). Write \(X\) as a sum of indicators and find \(E[X]\).

Solution

Let \(I_j=1\) when trial \(j\) succeeds for \(j\in\{2,3,5,7\}\). Then \(X=I_2+I_3+I_5+I_7\), so \(E[X]=4p\).

2. Concept: empty cells in a small allocation
ConceptOccupancy

Six balls are independently placed into four labelled boxes. Let \(X\) be the number of empty boxes. Compute \(E[X]\).

Solution

For box \(j\), let \(I_j=1\) if it is empty. A fixed box is missed by all six balls with probability \((3/4)^6\). Hence \(E[X]=4(3/4)^6\).

4. Concept: selected red cards
ConceptHypergeometric

A hand of \(k\) cards is drawn from a deck with \(r\) red cards and \(b\) black cards. Let \(X\) be the number of red cards in the hand. Use indicators to compute \(E[X]\).

Solution

Let \(I_j=1\) if red card \(j\) is selected. Since \(P(I_j=1)=k/(r+b)\), \(E[X]=r k/(r+b)\).

5. Concept: exact two visitors at a counter
ConceptIndex set

\(m\) visitors independently choose one of \(s\) counters uniformly. Let \(X\) be the number of counters receiving exactly two visitors. Compute \(E[X]\).

Solution

For counter \(j\), let \(I_j=1\) if exactly two visitors choose it. Then \(P(I_j=1)=\binom m2(1/s)^2(1-1/s)^{m-2}\), so \(E[X]=s\binom m2(1/s)^2(1-1/s)^{m-2}\).

Problems 6-8 - Integration Problems
6. Integration: matches after conditioning on a chosen subset
IntegrationConditioning

A random subset \(S\) of size \(k\) is chosen from \(\{1,\ldots,n\}\). Independently, a random permutation \(\pi\) of \(1,\ldots,n\) is chosen. Let \(X\) count elements \(i\in S\) with \(\pi(i)=i\). Find \(E[X\mid S]\) and \(E[X]\).

Solution

Given \(S\), write \(X=\sum_{i\in S}I_i\), where \(I_i=1\) if \(i\) is fixed. Each fixed element has probability \(1/n\), so \(E[X\mid S]=k/n\). Averaging gives \(E[X]=k/n\).

7. Integration: pair moment for adjacent heads
IntegrationPair terms

A fair coin is tossed \(n\ge4\) times. Let \(X\) be the number of adjacent pairs \(HH\). Express \(E[X]\) and describe the two pair types needed for \(E[X(X-1)]\).

Solution

Let \(I_i=1\) for \(HH\) at positions \(i,i+1\). Then \(E[X]=(n-1)/4\). For \(E[X(X-1)]\), ordered distinct pairs are either overlapping, such as \(i\) and \(i+1\), requiring three heads with probability \(1/8\), or disjoint, requiring four heads with probability \(1/16\).

8. Integration: accepted applications under a random pool
IntegrationTower

Given \(N\), each of \(N\) applications is accepted with probability \(q\), independently. \(N\) has mean \(\mu\). Let \(X\) be the number accepted. Compute \(E[X]\) and state the indicator decomposition.

Solution

Given \(N\), write \(X=\sum_{i=1}^N I_i\), with \(E[I_i\mid N]=q\). Thus \(E[X\mid N]=qN\), so \(E[X]=qE[N]=q\mu\).

Problems 9-10 - ISI-Style Challenge
9. ISI-style challenge: isolated occupied boxes
ISI-styleOccupancy

\(n\) balls are independently placed into \(n\) boxes arranged in a line. Let \(X\) be the number of boxes that contain exactly one ball and whose immediate neighbours, when they exist, are empty. Find \(E[X]\).

Solution

For an interior box, one specified box has exactly one ball and its two neighbours are empty with probability \(n(1/n)(1-3/n)^{n-1}\). There are \(n-2\) interior boxes. Each endpoint has one neighbour, giving probability \(n(1/n)(1-2/n)^{n-1}\). Hence \(E[X]=(n-2)(1-3/n)^{n-1}+2(1-2/n)^{n-1}\).

10. ISI-style challenge: double fixed ordered pairs
ISI-styleSecond moment

For a random permutation of \(n\ge3\), let \(X\) be the number of unordered pairs \(\{i,j\}\) such that both \(i\) and \(j\) are fixed. Compute \(E[X]\).

Solution

For each unordered pair \(\{i,j\}\), let \(I_{ij}=1\) if both are fixed. The probability is \((n-2)!/n!=1/[n(n-1)]\). There are \(\binom n2\) pairs, so \(E[X]=\binom n2/[n(n-1)]=1/2\).
