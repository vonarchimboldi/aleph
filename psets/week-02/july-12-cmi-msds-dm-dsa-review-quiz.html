11. Induction setup
MCQStrong induction

A proof of \(P(n)\) for \(n\ge2\) uses both \(P(n-1)\) and \(P(n-2)\). Which setup is sufficient? Select all correct options.

  1. Prove base cases \(P(0)\), \(P(1)\), then use strong induction.
  2. Prove only \(P(1)\), then use ordinary induction no matter what.
  3. Use strong induction and include enough base cases to cover the first transition.
  4. No base case is needed if the formula looks obvious.
