Core Pattern

Before calculating, ask which random variable makes the question easy after it is known.

\[ E[X]=E[E[X\mid Y]], \qquad \operatorname{Var}(X)=E[\operatorname{Var}(X\mid Y)] +\operatorname{Var}(E[X\mid Y]). \]

A useful conditioning variable often gives a simple binomial, geometric, uniform, or indicator-count problem inside.

Choice Checklist

  • Does one variable decide the sample size?
  • Does one variable decide the success probability?
  • Does one variable decide the range of another variable?
  • Does one variable make indicators independent?
  • Write the conditioning variable before the formula.
  • Compute the inner expectation in plain words first.
  • Only then average over the conditioning variable.
  • For variance, keep both total-variance terms.
Problems 1-5 · Mechanics
1. Choose the Easier First Card
Mechanics Conditioning choice 6 min

A card is drawn from a standard deck. If it is red, a fair coin is tossed \(4\) times. If it is black, the coin is tossed \(10\) times. Let \(H\) be the number of heads. Find \(E[H]\).

Solution

Condition on the card color. The color tells us the number of tosses.

\[ E[H\mid \text{red}]=4\cdot\frac12=2, \qquad E[H\mid \text{black}]=10\cdot\frac12=5. \]

Both colors have probability \(1/2\), so

\[ E[H]=\frac12(2)+\frac12(5)=\frac72. \]
2. Random Trial Count From a Die
Mechanics Random count 7 min

A fair die is rolled and shows \(K\). Then \(K+2\) independent Bernoulli trials are run, each with success probability \(p\). Let \(S\) be the number of successes. Find \(E[S]\).

Solution

Condition on \(K\), because \(K\) fixes the number of trials. If \(K=k\), then

\[ E[S\mid K=k]=(k+2)p. \]

Thus \(E[S\mid K]=(K+2)p\). Average over \(K\):

\[ E[S]=E[(K+2)p]=p(E[K]+2). \]

For a fair die, \(E[K]=7/2\). Therefore

\[ E[S]=\frac{11p}{2}. \]
3. Random Success Probability From a Switch
Mechanics Mixture 8 min

A switch chooses mode A with probability \(1/5\) and mode B with probability \(4/5\). In mode A, each of \(20\) messages is delivered with probability \(0.9\). In mode B, each is delivered with probability \(0.6\). Conditional on the mode, deliveries are independent. Let \(D\) be the number delivered. Find \(E[D]\).

Solution

Condition on the mode, since the mode decides the delivery probability.

\[ E[D\mid A]=20(0.9)=18, \qquad E[D\mid B]=20(0.6)=12. \]

Now average:

\[ E[D]=\frac15(18)+\frac45(12)=\frac{18+48}{5}=\frac{66}{5}. \]
4. Conditional Range of a Uniform Variable
Mechanics Random range 9 min

Let \(U\sim\operatorname{Unif}(0,1)\). Conditional on \(U=u\), let \(X\sim\operatorname{Unif}(u,1)\). Find \(E[X]\).

Solution

Condition on \(U\), because \(U\) decides the interval for \(X\). For fixed \(u\), the mean of a uniform variable on \([u,1]\) is

\[ E[X\mid U=u]=\frac{u+1}{2}. \]

So

\[ E[X]=E\left[\frac{U+1}{2}\right] =\frac{E[U]+1}{2}. \]

Since \(E[U]=1/2\),

\[ E[X]=\frac{3}{4}. \]
5. Two Possible Conditioning Variables
Mechanics Method choice 10 min

First choose a box: Box A with probability \(1/3\), Box B with probability \(2/3\). Box A gives a red ball with probability \(1/4\); Box B gives a red ball with probability \(3/5\). Then draw \(12\) times with replacement from the chosen box. Let \(R\) be the number of red draws. Find \(E[R]\), and state why conditioning on the box is better than conditioning on \(R\).

Solution

Condition on the box because the box decides the red probability. If Box A is chosen,

\[ E[R\mid A]=12\cdot\frac14=3. \]

If Box B is chosen,

\[ E[R\mid B]=12\cdot\frac35=\frac{36}{5}. \]

Average over the box:

\[ E[R]=\frac13(3)+\frac23\left(\frac{36}{5}\right) =1+\frac{24}{5} =\frac{29}{5}. \]

Conditioning on \(R\) is not useful because \(R\) is the final quantity we are trying to understand. The box is useful because it simplifies the inside probability.

Problems 6-8 · Applications
6. Variance Under a Random Mode
Application Total variance 16 min

A mode \(M\) is chosen. With probability \(1/2\), \(M=A\), and then \(X\sim\operatorname{Binomial}(10,1/5)\). With probability \(1/2\), \(M=B\), and then \(X\sim\operatorname{Binomial}(10,4/5)\). Find \(E[X]\) and \(\operatorname{Var}(X)\).

Solution

Condition on \(M\). The conditional means are

\[ E[X\mid A]=10\cdot\frac15=2, \qquad E[X\mid B]=10\cdot\frac45=8. \]

Thus

\[ E[X]=\frac12(2)+\frac12(8)=5. \]

The conditional variances are equal:

\[ \operatorname{Var}(X\mid A)=10\cdot\frac15\cdot\frac45=\frac85, \qquad \operatorname{Var}(X\mid B)=10\cdot\frac45\cdot\frac15=\frac85. \]

Use total variance:

\[ \operatorname{Var}(X)=E[\operatorname{Var}(X\mid M)] +\operatorname{Var}(E[X\mid M]). \]

The first term is \(8/5\). The second term is the variance of a variable that is \(2\) or \(8\), each with probability \(1/2\), whose mean is \(5\):

\[ \operatorname{Var}(E[X\mid M]) =\frac12(2-5)^2+\frac12(8-5)^2=9. \]

So

\[ \operatorname{Var}(X)=\frac85+9=\frac{53}{5}. \]
7. Indicators After Freezing the Environment
Application Indicators 17 min

A random variable \(P\) has \(E[P]=\mu\). Conditional on \(P\), events \(A_1,\ldots,A_n\) are independent and each has probability \(P\). Let \(X\) be the number of events that occur. Find \(E[X]\). Then explain why the independence after conditioning is not needed for the expectation.

Solution

Let \(I_i=\mathbf{1}_{A_i}\). Conditional on \(P\),

\[ E[I_i\mid P]=P. \]

Therefore

\[ E[X\mid P]=E\left[\sum_{i=1}^n I_i\mid P\right]=nP. \]

Average over \(P\):

\[ E[X]=E[nP]=n\mu. \]

Independence after conditioning would matter for products or variance. For expectation, linearity only needs the individual conditional expectations.

8. Conditional Geometric Waiting Time
Application Geometric 18 min

A random variable \(P\) takes values \(1/2\) and \(1/4\), each with probability \(1/2\). Conditional on \(P=p\), a sequence of independent trials is run until the first success, with success probability \(p\) on each trial. Let \(T\) be the number of trials needed. Find \(E[T]\).

Solution

Condition on \(P\), because \(P\) decides the geometric mean. If the success probability is \(p\), then

\[ E[T\mid P=p]=\frac1p. \]

So

\[ E[T]=E\left[\frac1P\right] =\frac12\cdot 2+\frac12\cdot 4=3. \]

Notice that this is not \(1/E[P]\). The tower property averages the conditional mean \(1/P\), not the parameter first.

Problems 9-10 · Hidden-Pattern Problems
9. Hidden Tower in a Random Sample Maximum
Hidden Random sample size 22 min

A random sample size \(N\) is chosen, where \(P(N=1)=P(N=2)=1/2\). Conditional on \(N=n\), draw \(n\) independent Uniform\((0,1)\) variables and let \(M\) be their maximum. Find \(E[M]\).

Solution

Condition on \(N\), because \(N\) decides which maximum problem we have. For \(n\) independent Uniform\((0,1)\) variables, the expected maximum is

\[ E[M\mid N=n]=\frac{n}{n+1}. \]

Therefore

\[ E[M]=\frac12\cdot\frac12+\frac12\cdot\frac23 =\frac14+\frac13 =\frac7{12}. \]

The hidden step is recognizing that the sample size is the useful conditioning variable.

10. Hidden Total Variance in a Prediction Error
Hidden Prediction 25 min

A random variable \(Y\) has \(E[Y]=0\) and \(\operatorname{Var}(Y)=\sigma^2\). Conditional on \(Y\), let \(X=Y+\varepsilon\), where \(E[\varepsilon\mid Y]=0\) and \(\operatorname{Var}(\varepsilon\mid Y)=1+Y^2\). Find \(E[X]\) and \(\operatorname{Var}(X)\).

Solution

Condition on \(Y\). Since \(X=Y+\varepsilon\),

\[ E[X\mid Y]=Y+E[\varepsilon\mid Y]=Y. \]

So

\[ E[X]=E[E[X\mid Y]]=E[Y]=0. \]

For variance, use total variance:

\[ \operatorname{Var}(X)=E[\operatorname{Var}(X\mid Y)] +\operatorname{Var}(E[X\mid Y]). \]

Given \(Y\), the only random part of \(X\) is \(\varepsilon\), so

\[ \operatorname{Var}(X\mid Y)=1+Y^2. \]

Also \(E[X\mid Y]=Y\). Therefore

\[ \operatorname{Var}(X)=E[1+Y^2]+\operatorname{Var}(Y) =1+\sigma^2+\sigma^2 =1+2\sigma^2. \]
