Core Pattern

Simple least squares fits a line \( \widehat y=a+bx \) by making squared residuals small. For one predictor, the slope is a covariance-over-variance ratio.

\[ \widehat b=\frac{\sum_i (x_i-\overline x)(y_i-\overline y)}{\sum_i (x_i-\overline x)^2}, \qquad \widehat a=\overline y-\widehat b\,\overline x. \]

The fitted line passes through the mean point \((\overline x,\overline y)\). Residuals are observed minus fitted values: \(e_i=y_i-\widehat y_i\).

Setup Checklist

  • Compute \(\overline x\) and \(\overline y\) first.
  • Center each value: \(x_i-\overline x\) and \(y_i-\overline y\).
  • Compute the slope as centered cross-product divided by centered \(x\)-square.
  • Use the mean point to get the intercept.
  • Fitted value: \(\widehat y_i=\widehat a+\widehat b x_i\).
  • Residual: \(e_i=y_i-\widehat y_i\).
  • In simple OLS with intercept, residuals sum to zero.
  • The matrix equation \(X^TX\widehat\beta=X^Ty\) comes later; today is the meaning of each quantity.
Problems 1-5 · Concept Builders
1. Mean Point of a Small Dataset
Concept Means 6 min

For the data points \((1,2),(2,4),(3,5)\), compute \(\overline x\) and \(\overline y\). State the mean point.

Solution

The average of the \(x\)-values is

\[ \overline x=\frac{1+2+3}{3}=2. \]

The average of the \(y\)-values is

\[ \overline y=\frac{2+4+5}{3}=\frac{11}{3}. \]

The mean point is

\[ \left(2,\frac{11}{3}\right). \]
2. Centered Cross-Products
Concept Covariance numerator 8 min

Using the same points \((1,2),(2,4),(3,5)\), compute \(\sum_i (x_i-\overline x)(y_i-\overline y)\).

Solution

From Problem 1, \(\overline x=2\) and \(\overline y=11/3\). The centered values are

\[ x_i-\overline x:\ -1,0,1. \]

For \(y\), the centered values are

\[ 2-\frac{11}{3}=-\frac53,\quad 4-\frac{11}{3}=\frac13,\quad 5-\frac{11}{3}=\frac43. \]

The cross-products sum to

\[ (-1)\left(-\frac53\right)+0\left(\frac13\right)+1\left(\frac43\right) =\frac53+\frac43=3. \]
3. Simple OLS Slope
Concept Slope 8 min

For the data points \((1,2),(2,4),(3,5)\), use the centered formula to compute the OLS slope \(\widehat b\).

Solution

From Problem 2, the numerator is \(3\). The denominator is

\[ \sum_i (x_i-\overline x)^2=(-1)^2+0^2+1^2=2. \]

Therefore

\[ \widehat b=\frac{3}{2}. \]
4. Intercept From the Mean Point
Concept Intercept 7 min

For the same dataset, use \(\widehat b=3/2\), \(\overline x=2\), and \(\overline y=11/3\) to find the intercept \(\widehat a\).

Solution

The fitted line with intercept passes through \((\overline x,\overline y)\), so

\[ \widehat a=\overline y-\widehat b\,\overline x. \]

Substitute the values:

\[ \widehat a=\frac{11}{3}-\frac32\cdot2 =\frac{11}{3}-3 =\frac23. \]

The fitted line is

\[ \widehat y=\frac23+\frac32x. \]
5. Fitted Values and Residuals
Concept Residuals 9 min

For the fitted line \(\widehat y=2/3+(3/2)x\) and data \((1,2),(2,4),(3,5)\), compute the three residuals and check their sum.

Solution

The fitted values are

\[ \widehat y_1=\frac23+\frac32=\frac{13}{6}, \quad \widehat y_2=\frac23+3=\frac{11}{3}, \quad \widehat y_3=\frac23+\frac92=\frac{31}{6}. \]

The residuals \(e_i=y_i-\widehat y_i\) are

\[ 2-\frac{13}{6}=-\frac16,\quad 4-\frac{11}{3}=\frac13,\quad 5-\frac{31}{6}=-\frac16. \]

Their sum is

\[ -\frac16+\frac13-\frac16=0. \]
Problems 6-8 · Applications
6. Regression Through a Perfect Line
Application Perfect fit 10 min

The data points are \((1,3),(2,5),(3,7),(4,9)\). Find the OLS line without doing a long table.

Solution

Each time \(x\) increases by \(1\), \(y\) increases by \(2\). The points lie on a line with slope \(2\).

Use one point, say \((1,3)\), to find the intercept:

\[ 3=a+2(1). \]

So \(a=1\). The OLS line is

\[ \widehat y=1+2x. \]

Because all points are exactly on this line, every residual is zero.

7. Slope From Summary Sums
Application Summary statistics 12 min

For a dataset, suppose \(\sum_i (x_i-\overline x)(y_i-\overline y)=18\) and \(\sum_i (x_i-\overline x)^2=12\). Also suppose \(\overline x=4\) and \(\overline y=10\). Find the OLS line.

Solution

The slope is

\[ \widehat b=\frac{18}{12}=\frac32. \]

The intercept is

\[ \widehat a=\overline y-\widehat b\,\overline x =10-\frac32\cdot4 =10-6=4. \]

So the OLS line is

\[ \widehat y=4+\frac32x. \]
8. Residual Orthogonality to the Intercept
Application Residual sum 13 min

A fitted line with intercept gives fitted values \(\widehat y_1,\ldots,\widehat y_n\) and residuals \(e_i=y_i-\widehat y_i\). Explain why the residuals should sum to zero in an OLS fit with intercept.

Solution

OLS chooses \(a\) and \(b\) to minimize

\[ S(a,b)=\sum_i (y_i-a-bx_i)^2. \]

Differentiate with respect to the intercept \(a\):

\[ \frac{\partial S}{\partial a} = -2\sum_i (y_i-a-bx_i). \]

At the minimum this derivative is zero, so

\[ \sum_i (y_i-\widehat a-\widehat b x_i)=0. \]

The terms inside the sum are residuals. Hence

\[ \sum_i e_i=0. \]
Problems 9-10 · Hidden-Pattern / ISI Style
9. First Normal Equation
Challenge Normal equations 16 min

For simple regression with intercept, show that minimizing \(\sum_i(y_i-a-bx_i)^2\) gives the equation \(\sum_i y_i=n a+b\sum_i x_i\).

Solution

Let

\[ S(a,b)=\sum_i(y_i-a-bx_i)^2. \]

Differentiate with respect to \(a\):

\[ \frac{\partial S}{\partial a} = -2\sum_i(y_i-a-bx_i). \]

Set this equal to zero:

\[ \sum_i(y_i-a-bx_i)=0. \]

Expand the sum:

\[ \sum_i y_i-\sum_i a-b\sum_i x_i=0. \]

Since \(\sum_i a=na\), this becomes

\[ \sum_i y_i=n a+b\sum_i x_i. \]
10. Prediction and Residual Check
ISI-style Interpretation 18 min

A fitted regression line is \(\widehat y=5+0.8x\). A point has \(x=10\) and observed value \(y=14\). Find the fitted value and residual. Then say whether this point lies above or below the fitted line.

Solution

The fitted value is

\[ \widehat y=5+0.8(10)=13. \]

The residual is observed minus fitted:

\[ e=y-\widehat y=14-13=1. \]

The residual is positive, so the observed point lies above the fitted line.

Answer Summary

  1. Mean point \((2,11/3)\).
  2. Centered cross-product sum \(=3\).
  3. Slope \(\widehat b=3/2\).
  4. Intercept \(\widehat a=2/3\); fitted line \(\widehat y=2/3+(3/2)x\).
  5. Residuals \(-1/6,1/3,-1/6\), summing to \(0\).
  6. \(\widehat y=1+2x\).
  7. \(\widehat y=4+(3/2)x\).
  8. Residuals sum to zero because the intercept derivative sets \(\sum_i e_i=0\).
  9. The first normal equation is \(\sum_i y_i=n a+b\sum_i x_i\).
  10. Fitted value \(13\), residual \(1\), point above the line.
