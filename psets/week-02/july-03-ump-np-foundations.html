Core Pattern

The Neyman-Pearson idea starts with simple hypotheses. A test should reject \(H_0\) when the observed data is much more likely under \(H_1\) than under \(H_0\).

\[ \Lambda(x)=\frac{f_1(x)}{f_0(x)}. \]

Large \(\Lambda(x)\) favors \(H_1\). After finding the rejection region, choose its cutoff so that the probability of rejection under \(H_0\) is the required size.

Setup Checklist

  • Write \(H_0\) and \(H_1\) before doing algebra.
  • Write the likelihood under each hypothesis.
  • Form \(f_1/f_0\), then decide whether large or small observations favor \(H_1\).
  • Choose the cutoff using probability under \(H_0\), not under \(H_1\).
  • Size is \(P_{H_0}(\text{reject }H_0)\).
  • Power is \(P_{H_1}(\text{reject }H_0)\).
  • For discrete data, exact size may require a conservative cutoff or randomization.
  • Composite and monotone-likelihood-ratio tests come later; today is simple-vs-simple fluency.
Problems 1-5 · Concept Builders
1. State Size and Power
Concept Definitions 6 min

A test rejects \(H_0\) when an observation falls in a region \(R\). Write the size and power of this test when \(H_0:f=f_0\) and \(H_1:f=f_1\) are simple hypotheses.

Solution

The size is the probability of rejecting \(H_0\) when \(H_0\) is true:

\[ \alpha=P_0(X\in R). \]

The power is the probability of rejecting \(H_0\) when \(H_1\) is true:

\[ \pi=P_1(X\in R). \]

The same rejection region is measured under two different distributions. Size uses \(H_0\); power uses \(H_1\).

2. Likelihood Ratio Direction for Bernoulli
Concept Direction 7 min

Let \(X\sim\operatorname{Bernoulli}(p)\). Test \(H_0:p=0.3\) against \(H_1:p=0.7\). Compute \(f_1(x)/f_0(x)\) for \(x=0,1\), and say which value of \(X\) favors \(H_1\).

Solution

For \(x=1\),

\[ \frac{f_1(1)}{f_0(1)}=\frac{0.7}{0.3}=\frac{7}{3}. \]

For \(x=0\),

\[ \frac{f_1(0)}{f_0(0)}=\frac{0.3}{0.7}=\frac{3}{7}. \]

The likelihood ratio is larger when \(X=1\). So \(X=1\) favors \(H_1\).

3. Normal Mean Direction
Concept Normal shift 8 min

Let \(X\sim N(\mu,1)\). Test \(H_0:\mu=0\) against \(H_1:\mu=1\). Without choosing a cutoff yet, show that large \(X\) values favor \(H_1\).

Solution

The likelihood ratio is

\[ \frac{f_1(x)}{f_0(x)} = \frac{\exp\{-(x-1)^2/2\}}{\exp\{-x^2/2\}}. \]

Simplify the exponent:

\[ -\frac{(x-1)^2}{2}+\frac{x^2}{2} = x-\frac12. \]

So

\[ \frac{f_1(x)}{f_0(x)}=\exp\left(x-\frac12\right). \]

This increases as \(x\) increases. Large observations favor \(H_1\).

4. Size Cutoff for a Standard Normal Test
Concept Cutoff 8 min

For \(X\sim N(\mu,1)\), test \(H_0:\mu=0\) against \(H_1:\mu=1\) by rejecting for \(X>c\). Choose \(c\) so the size is \(0.05\). You may write the answer using the standard normal quantile \(z_{0.95}\).

Solution

Size is computed under \(H_0\). Under \(H_0\), \(X\sim N(0,1)\). We need

\[ P_0(X>c)=0.05. \]

Thus \(c\) is the 95th percentile of the standard normal distribution:

\[ c=z_{0.95}. \]

Numerically this is about \(1.645\), but the quantile form is enough.

5. Power After Choosing the Cutoff
Concept Power 9 min

Continue Problem 4. With \(c=z_{0.95}\), write the power of the test under \(H_1:\mu=1\).

Solution

Power is computed under \(H_1\). Under \(H_1\), \(X\sim N(1,1)\). The power is

\[ P_1(X>z_{0.95}). \]

Standardize using \(Z=X-1\sim N(0,1)\):

\[ P_1(X>z_{0.95})=P(Z>z_{0.95}-1). \]

Equivalently,

\[ \text{Power}=1-\Phi(z_{0.95}-1). \]
Problems 6-8 · Applications
6. One Observation From an Exponential Model
Application Exponential 13 min

Let \(X\sim\operatorname{Exponential}(\lambda)\) with density \(\lambda e^{-\lambda x}\), \(x\ge0\). Test \(H_0:\lambda=1\) against \(H_1:\lambda=2\). Find whether small or large \(X\) favors \(H_1\).

Solution

The likelihood ratio is

\[ \frac{f_1(x)}{f_0(x)} = \frac{2e^{-2x}}{e^{-x}} = 2e^{-x}. \]

This decreases as \(x\) increases. It is largest for small \(x\).

So small observations favor \(H_1:\lambda=2\). This makes sense because a larger exponential rate has a smaller mean waiting time.

8. Binomial Cutoff Without Randomization
Application Discrete size 15 min

Let \(X\sim\operatorname{Binomial}(4,p)\). Test \(H_0:p=0.25\) against \(H_1:p=0.75\). The likelihood ratio favors large \(X\). If the test rejects for \(X\ge3\), compute its size and power.

Solution

The size is \(P_{0.25}(X\ge3)\):

\[ P_{0.25}(X\ge3) = \binom43(0.25)^3(0.75)+(0.25)^4. \]

This is

\[ 4\cdot\frac{1}{64}\cdot\frac34+\frac{1}{256} = \frac{12}{256}+\frac{1}{256} = \frac{13}{256}. \]

The power is \(P_{0.75}(X\ge3)\):

\[ \binom43(0.75)^3(0.25)+(0.75)^4 = \frac{108}{256}+\frac{81}{256} = \frac{189}{256}. \]
Problems 9-10 · Hidden-Pattern / ISI Style
9. Normal Sample Mean Test
Challenge Sample mean 18 min

Let \(X_1,\ldots,X_n\) be independent \(N(\mu,1)\). Test \(H_0:\mu=0\) against \(H_1:\mu=1\). Show that the likelihood-ratio test rejects for large \(\overline X\), and give the size \(0.05\) cutoff.

Solution

The likelihood ratio is

\[ \frac{L_1}{L_0} = \exp\left(\sum_{i=1}^n x_i-\frac{n}{2}\right). \]

This increases as \(\sum_i x_i\) increases, so it also increases as \(\overline X\) increases. The test rejects for large \(\overline X\).

Under \(H_0\), \(\overline X\sim N(0,1/n)\). For size \(0.05\), choose \(c\) so

\[ P_0(\overline X>c)=0.05. \]

Standardizing gives

\[ P(Z>\sqrt n\,c)=0.05. \]

Hence

\[ c=\frac{z_{0.95}}{\sqrt n}. \]
10. Randomization at a Boundary Point
ISI-style Discrete test 20 min

Let \(X\sim\operatorname{Binomial}(2,p)\). Test \(H_0:p=0.5\) against \(H_1:p=0.9\). Since large \(X\) favors \(H_1\), consider rejecting always when \(X=2\), rejecting with probability \(\gamma\) when \(X=1\), and not rejecting when \(X=0\). Choose \(\gamma\) so the size is \(0.30\), then compute the power.

Solution

Under \(H_0:p=0.5\),

\[ P_0(X=2)=0.25, \qquad P_0(X=1)=0.50. \]

The size is

\[ 0.25+\gamma(0.50). \]

Set this equal to \(0.30\):

\[ 0.25+0.50\gamma=0.30. \]

Thus

\[ \gamma=0.10. \]

Under \(H_1:p=0.9\),

\[ P_1(X=2)=0.81, \qquad P_1(X=1)=2(0.9)(0.1)=0.18. \]

The power is

\[ 0.81+0.10(0.18)=0.828. \]

Answer Summary

  1. Size \(=P_0(X\in R)\); power \(=P_1(X\in R)\).
  2. \(X=1\) favors \(H_1:p=0.7\).
  3. \(f_1/f_0=\exp(x-1/2)\), so large \(X\) favors \(H_1\).
  4. \(c=z_{0.95}\).
  5. Power \(=1-\Phi(z_{0.95}-1)\).
  6. Small \(X\) favors \(H_1:\lambda=2\).
  7. \(c=-\log(0.90)\), power \(=0.19\).
  8. Size \(=13/256\), power \(=189/256\).
  9. Reject for \(\overline X>z_{0.95}/\sqrt n\).
  10. \(\gamma=0.10\), power \(=0.828\).
