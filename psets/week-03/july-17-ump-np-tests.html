Core Pattern

For simple-vs-simple hypotheses, reject \(H_0\) when the alternative likelihood is large relative to the null likelihood.

\[\Lambda(x)=\frac{L_1(x)}{L_0(x)},\qquad \text{reject }H_0\text{ for large }\Lambda(x).\]

Setup Checklist

  • Write null and alternative distributions.
  • Compute the likelihood ratio in one direction and keep it fixed.
  • Convert the ratio inequality into a statistic inequality.
  • Set the cutoff using probability under \(H_0\).
  • Compute power using probability under \(H_1\).
  • For UMP, look for monotone likelihood ratio.
Problems 1-5 - Concept Builders
1. Identify the null distribution
ConceptHypotheses

A coin is tested with \(H_0:p=1/2\) against \(H_1:p=3/4\). For \(n\) tosses, what is the null distribution of \(X\), the number of heads?

Solution

Under \(H_0\), \(X\sim\operatorname{Binomial}(n,1/2)\). Size is computed using this distribution.

2. Likelihood ratio direction for binomial data
ConceptRatio

For \(H_0:p=1/2\) and \(H_1:p=3/4\), write \(L_1/L_0\) as a function of \(X=x\).

Solution

The ratio is \(\frac{(3/4)^x(1/4)^{n-x}}{(1/2)^n}=(3^x)/(2^n)\). It increases with \(x\), so large head counts support \(H_1\).

3. Size of a one-tail binomial test
ConceptSize

For \(n=4\), reject \(H_0:p=1/2\) when \(X=4\). Compute the size.

Solution

Size is the rejection probability under \(H_0\). Thus \(\alpha=P_{1/2}(X=4)=(1/2)^4=1/16\).

4. Power of the same binomial test
ConceptPower

For the test in Problem 3, compute the power at \(p=3/4\).

Solution

Power is rejection probability under the alternative. It equals \(P_{3/4}(X=4)=(3/4)^4=81/256\).

5. Normal mean rejection direction
ConceptDirection

For one observation \(X\sim N(\mu,1)\), test \(H_0:\mu=0\) against \(H_1:\mu=1\). Should the NP rejection region use large or small \(X\)?

Solution

The alternative mean is larger. The likelihood ratio increases with \(X\), so the rejection region uses large values of \(X\).

Problems 6-8 - Integration Problems
6. Normal cutoff from size
IntegrationCutoff

For \(X\sim N(\mu,1)\), test \(H_0:\mu=0\) against \(H_1:\mu=1\). Reject when \(X>c\). Choose \(c\) for size \(0.05\).

Solution

Under \(H_0\), \(X\sim N(0,1)\). We need \(P_0(X>c)=0.05\), so \(c=z_{0.95}\), about \(1.645\).

7. Normal power at the alternative
IntegrationPower

Using the cutoff \(c=1.645\) from Problem 6, write the power at \(\mu=1\).

Solution

Under \(\mu=1\), \(X-1\sim N(0,1)\). Power is \(P_1(X>1.645)=P(Z>0.645)=1-\Phi(0.645)\).

8. Discrete boundary randomization
IntegrationRandomization

Under \(H_0\), \(X\sim\operatorname{Binomial}(2,1/2)\). Reject when \(X=2\), and reject with probability \(\gamma\) when \(X=1\). Choose \(\gamma\) for size \(1/2\).

Solution

Under \(H_0\), \(P(X=2)=1/4\) and \(P(X=1)=1/2\). Size is \(1/4+\gamma/2\). Set \(1/4+\gamma/2=1/2\), so \(\gamma=1/2\).

Problems 9-10 - ISI-Style Challenge
9. Exponential rate likelihood ratio
ISI-styleLR

For iid exponential data with rate \(\lambda\), test \(H_0:\lambda=1\) against \(H_1:\lambda=2\). In terms of \(S=\sum_iX_i\), does the NP test reject for large or small \(S\)?

Solution

The ratio \(L_1/L_0=2^n e^{-S}\). This decreases as \(S\) increases, so the test rejects for small \(S\). Larger rate means smaller observations.

10. Monotone likelihood ratio cue
ISI-styleUMP

For binomial \(X\sim\operatorname{Binomial}(n,p)\), explain why tests of \(H_0:p\le p_0\) against \(H_1:p>p_0\) use large \(X\).

Solution

For \(p_1>p_0\), the likelihood ratio is proportional to \(\left[\frac{p_1(1-p_0)}{p_0(1-p_1)}\right]^X\), and the base is greater than 1. The ratio increases with \(X\), so the one-sided UMP test rejects for large \(X\), calibrated at the boundary \(p_0\).

Answer Summary

  2. \(3^x/2^n\)
  3. \(1/16\)
  4. \(81/256\)
  5. Large \(X\)
  6. \(z_{0.95}\approx1.645\)
  7. \(1-\Phi(0.645)\)
  8. \(\gamma=1/2\)
  9. Small \(S\)
  10. Large \(X\), calibrated at \(p_0\)
