Core Pattern

When a problem asks for the expected number of objects with a property, write the count as a sum of indicators.

\[X=\sum_i I_i,\qquad I_i=\mathbf{1}_{A_i},\qquad E[X]=\sum_i P(A_i).\]

For variance, expand pair terms and classify whether two indicator events overlap, exclude each other, or can occur together.

Setup Checklist

  • Name the object being counted.
  • Choose one indicator for one atomic object.
  • Use symmetry only after checking all indicators have the same probability.
  • For \(X^2\), separate identical and distinct pairs.
  • For \(X(X-1)\), count ordered distinct pairs.
  • Write one correction note for every missed setup.
Problems 1-5 - Concept Builders
1. Marked successes in eight trials
ConceptLinearity

Eight independent trials each succeed with probability \(p\). Let \(X\) be the number of successes in positions \(2,4,6,8\). Write \(X\) as a sum of indicators and compute \(E[X]\).

Solution

Let \(I_j=1\) if trial \(j\) succeeds, for \(j\in\{2,4,6,8\}\). Then \(X=I_2+I_4+I_6+I_8\), so \(E[X]=4p\).

2. Fixed colours on labelled cards
ConceptSymmetry

A random permutation of \(n\) labelled cards is placed into \(n\) labelled slots. Let \(X\) be the number of cards in their original slots. Find \(E[X]\).

Solution

Let \(I_i=1\) if card \(i\) lands in slot \(i\). Then \(P(I_i=1)=1/n\). Therefore \(E[X]=\sum_{i=1}^n 1/n=1\).

3. Nonempty rooms after choices
ConceptOccupancy

\(m\) students independently choose one of \(r\) rooms uniformly. Let \(X\) be the number of nonempty rooms. Compute \(E[X]\).

Solution

Let \(I_j=1\) if room \(j\) is nonempty. One room is empty with probability \((1-1/r)^m\), so \(P(I_j=1)=1-(1-1/r)^m\). Hence \(E[X]=r[1-(1-1/r)^m]\).

4. Adjacent equal coin pairs
ConceptDependence

A fair coin is tossed \(n\) times. Let \(X\) be the number of adjacent pairs with equal outcomes. Compute \(E[X]\).

Solution

For \(1\le i\le n-1\), let \(I_i=1\) if tosses \(i\) and \(i+1\) match. \(P(I_i=1)=1/2\). Thus \(E[X]=(n-1)/2\). The adjacent indicators are dependent, but linearity still works.

5. Exactly one occupied box
ConceptIndex set

\(n\) balls are independently thrown into \(n\) labelled boxes. Let \(X\) be the number of boxes containing exactly one ball. Compute \(E[X]\).

Solution

Let \(I_j=1\) if box \(j\) has exactly one ball. For a fixed box, choose the ball that lands there and force all other balls away: \(P(I_j=1)=n(1/n)(1-1/n)^{n-1}\). Hence \(E[X]=n(1-1/n)^{n-1}\).

Problems 6-8 - Integration Problems
6. Variance of fixed points with pair terms
IntegrationVariance

For a random permutation of \(n\ge2\) objects, let \(X\) be the number of fixed points. Compute \(E[X(X-1)]\) and \(\operatorname{Var}(X)\).

Solution

Let \(I_i=1\) if \(i\) is fixed. For \(i\ne j\), \(P(I_iI_j=1)=1/[n(n-1)]\). Thus \(E[X(X-1)]=n(n-1)/[n(n-1)]=1\). Since \(E[X]=1\), \(E[X^2]=E[X(X-1)]+E[X]=2\), so \(\operatorname{Var}(X)=1\).

7. Random subset intersections
IntegrationExchangeability

A subset \(A\) of size \(a\) and a subset \(B\) of size \(b\) are chosen uniformly from \(\{1,\ldots,n\}\), independently. Let \(X=|A\cap B|\). Compute \(E[X]\).

Solution

Let \(I_i=1\) if element \(i\) lies in both \(A\) and \(B\). Then \(P(i\in A)=a/n\), \(P(i\in B)=b/n\), and the choices are independent across the two sets. Hence \(P(I_i=1)=ab/n^2\), so \(E[X]=ab/n\).

8. Conditioning on a random number of visitors
IntegrationTower

Given \(N\), each of \(N\) visitors independently buys a ticket with probability \(p\). Let \(X\) be the number of ticket buyers. Show that \(E[X]=pE[N]\).

Solution

Condition on \(N\). Given \(N\), \(X\) is binomial with mean \(pN\). Therefore \(E[X\mid N]=pN\), and by the tower property \(E[X]=E[pN]=pE[N]\).

10. Empty boxes with a second moment
ISI-styleOccupancy

\(n\ge2\) balls are independently placed into \(n\) labelled boxes. Let \(X\) be the number of empty boxes. Compute \(E[X]\) and \(E[X(X-1)]\).

Solution

Let \(I_j=1\) if box \(j\) is empty. \(E[X]=n(1-1/n)^n\). For ordered \(j\ne k\), both boxes are empty with probability \((1-2/n)^n\). Therefore \(E[X(X-1)]=n(n-1)(1-2/n)^n\).
