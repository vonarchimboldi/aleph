Core Pattern

With an intercept, the fitted line passes through \((\overline x,\overline y)\), and the slope is a centered cross-product divided by a centered square.

\[\widehat b=\frac{\sum_i(x_i-\overline x)(y_i-\overline y)}{\sum_i(x_i-\overline x)^2},\qquad \widehat a=\overline y-\widehat b\overline x.\]

Setup Checklist

  • Compute means before slope.
  • Use centered quantities for slope.
  • Check residuals sum to zero when an intercept is present.
  • For matrix OLS, write \(X^TX\widehat\beta=X^Ty\).
  • Residuals are orthogonal to columns of \(X\).
  • Interpret coefficients in one sentence.
Problems 1-5 - Concept Builders
1. Mean point for July regression data
ConceptMeans

For \((1,1),(2,3),(3,5)\), compute \((\overline x,\overline y)\).

Solution

\(\overline x=2\) and \(\overline y=3\). The mean point is \((2,3)\).

2. Centered slope numerator
ConceptCross-product

For \((1,1),(2,3),(3,5)\), compute \(\sum_i(x_i-\overline x)(y_i-\overline y)\).

Solution

Centered \(x\) values are \(-1,0,1\), and centered \(y\) values are \(-2,0,2\). The sum is \(2+0+2=4\).

3. Slope and intercept for exact line
ConceptLine fit

Use Problems 1-2 to fit \(\widehat y=a+bx\) for \((1,1),(2,3),(3,5)\).

Solution

The denominator is \((-1)^2+0^2+1^2=2\), so \(\widehat b=4/2=2\). Then \(\widehat a=3-2\cdot2=-1\). The line is \(\widehat y=-1+2x\).

4. Residual sum check
ConceptResiduals

For the fitted line \(\widehat y=-1+2x\) on the three data points above, compute the residuals.

Solution

The fitted values are \(1,3,5\). The residuals are \(0,0,0\), so their sum is zero.

5. Normal equation dimensions
ConceptMatrix setup

If \(X\) is an \(n\times2\) design matrix for intercept plus one feature, what are the dimensions of \(X^TX\) and \(X^Ty\)?

Solution

\(X^TX\) is \(2\times2\), and \(X^Ty\) is \(2\times1\). The coefficient vector \(\widehat\beta\) has two entries.

Problems 6-8 - Application Problems
6. Fit from summary statistics
ApplicationSummary

Given \(\overline x=4\), \(\overline y=10\), \(\sum(x_i-\overline x)(y_i-\overline y)=30\), and \(\sum(x_i-\overline x)^2=6\), find the fitted line.

Solution

The slope is \(30/6=5\). The intercept is \(10-5\cdot4=-10\). The fitted line is \(\widehat y=-10+5x\).

7. Residual orthogonality statement
ApplicationGeometry

In matrix OLS with full column rank, state the normal-equation residual orthogonality condition.

Solution

The residual vector is \(e=y-X\widehat\beta\). The normal equations give \(X^Te=0\), meaning the residual is orthogonal to every column of the design matrix.

8. Scaling the response
ApplicationInvariance

If every \(y_i\) is multiplied by 3 while \(x_i\) is unchanged, how do the simple-regression slope and intercept change?

Solution

The centered cross-product and \(\overline y\) are multiplied by 3, while the \(x\)-denominator is unchanged. Both slope and intercept are multiplied by 3.

Problems 9-10 - ISI-Style Challenge
9. Centered predictor intercept shortcut
ISI-styleShortcut

Suppose \(\overline x=0\) in a simple regression with intercept. What is \(\widehat a\)?

Solution

Since the fitted line passes through \((\overline x,\overline y)\), \(\widehat a=\overline y-\widehat b\overline x=\overline y\).

10. Hat matrix idempotence cue
ISI-styleProjection

For full-column-rank \(X\), the OLS fitted vector is \(\widehat y=Hy\), where \(H=X(X^TX)^{-1}X^T\). State two algebraic properties of \(H\).

Solution

The matrix \(H\) is symmetric and idempotent: \(H^T=H\) and \(H^2=H\). These encode orthogonal projection onto the column space of \(X\).
