Questions 1-6 - Weekly ISI-Style Review
1. Method of indicators review: occupied singletons
ReviewMethod of indicators

\(n\) balls are thrown into \(n\) labelled boxes. Let \(X\) be the number of boxes with exactly one ball. Write \(X\) as a sum of indicators, compute \(E[X]\), and state the main pair-term issue if asked for variance.

2. Conditional expectation review: random coins
ReviewTower

Roll a fair die \(Y\), then toss \(Y\) fair coins. Let \(X\) be the number of heads. Compute \(E[X]\) and name the conditioning variable.

3. Order statistics review: third value threshold
ReviewOrder statistics

For seven iid observations with CDF \(F\), express \(P(X_{(3)}\le a)\).

4. MLE review: support-dependent endpoint
ReviewMLE

\(X_1,\ldots,X_n\) are iid Uniform\((0,\theta)\). Write the likelihood with support and find the MLE.

5. UMP review: binomial right tail
ReviewUMP

For \(X\sim\operatorname{Binomial}(n,p)\), explain why testing \(H_0:p\le p_0\) against \(H_1:p>p_0\) uses a large-\(X\) rejection region.

6. OLS review: residual orthogonality
ReviewOLS

In matrix OLS, \(\widehat\beta\) solves \(X^TX\widehat\beta=X^Ty\). Show the residual vector is orthogonal to the columns of \(X\).
