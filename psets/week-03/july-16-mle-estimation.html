Core Pattern

The likelihood is the joint probability or density of the observed data as a function of the parameter.

\[L(\theta)=\prod_{i=1}^n f_\theta(x_i),\qquad \ell(\theta)=\log L(\theta).\]

Only drop factors that do not depend on the parameter. Check support restrictions before trusting a derivative.

Setup Checklist

  • State the model and parameter space.
  • Write the joint likelihood.
  • Take logs only after preserving parameter-dependent terms.
  • Solve first-order equations when the support is fixed.
  • Check boundaries and support-dependent constraints.
  • For estimator quality, compute bias or MSE from the sampling law.
Problems 1-5 - Concept Builders
1. Bernoulli likelihood setup
ConceptLikelihood

Data \(x_1,\ldots,x_n\in\{0,1\}\) come from Bernoulli\((p)\). Write \(L(p)\).

Solution

Let \(s=\sum_i x_i\). The likelihood is \(L(p)=p^s(1-p)^{n-s}\), for \(0\le p\le1\).

2. Bernoulli maximum likelihood estimate
ConceptMLE

Using the likelihood from Problem 1, find the MLE of \(p\) when \(0

Solution

The log-likelihood is \(s\log p+(n-s)\log(1-p)\). Differentiating gives \(s/p-(n-s)/(1-p)=0\), so \(\widehat p=s/n\).

3. Poisson likelihood kernel
ConceptKernel

\(X_1,\ldots,X_n\) are iid Poisson\((\lambda)\). Write the part of the likelihood depending on \(\lambda\).

Solution

The joint likelihood is \(\prod_i e^{-\lambda}\lambda^{x_i}/x_i!\). The parameter-dependent part is \(e^{-n\lambda}\lambda^{\sum_i x_i}\).

4. Poisson MLE
ConceptDerivative

For iid Poisson\((\lambda)\) data, find \(\widehat\lambda_{\mathrm{MLE}}\).

Solution

The log-likelihood kernel is \(-n\lambda+s\log\lambda\), where \(s=\sum_i x_i\). Differentiating gives \(-n+s/\lambda=0\), so \(\widehat\lambda=s/n=\overline X\).

5. Uniform upper-bound support
ConceptSupport

\(X_1,\ldots,X_n\) are iid Uniform\((0,\theta)\). Write the likelihood including support.

Solution

The density product is \(\theta^{-n}\) only if \(0\le x_i\le\theta\) for all \(i\). Hence \(L(\theta)=\theta^{-n}\mathbf{1}\{\theta\ge x_{(n)}\}\), with \(\theta>0\).

Problems 6-8 - Integration Problems
6. Uniform endpoint MLE
IntegrationBoundary

For iid Uniform\((0,\theta)\) data, find the MLE of \(\theta\).

Solution

The likelihood is \(\theta^{-n}\) for \(\theta\ge x_{(n)}\). This decreases as \(\theta\) increases, so the maximum occurs at the smallest allowed value: \(\widehat\theta=x_{(n)}\).

7. Exponential rate estimate
IntegrationRate

\(X_1,\ldots,X_n\) are iid exponential with rate \(\lambda\). Find the MLE of \(\lambda\).

Solution

The likelihood kernel is \(\lambda^n e^{-\lambda\sum x_i}\). The log kernel is \(n\log\lambda-\lambda\sum x_i\). Differentiating gives \(\widehat\lambda=n/\sum_i x_i=1/\overline X\).

8. Bias of the Bernoulli MLE
IntegrationBias

For Bernoulli\((p)\), the MLE is \(\widehat p=\overline X\). Compute its bias.

Solution

Since \(E[\overline X]=p\), the bias is \(E[\widehat p]-p=0\). The sample proportion is unbiased.

Problems 9-10 - ISI-Style Challenge
9. Shifted uniform lower endpoint
ISI-styleSupport

\(X_1,\ldots,X_n\) are iid Uniform\((\theta,\theta+1)\). Find the set of \(\theta\) values with positive likelihood.

Solution

Positive likelihood requires \(\theta\le x_i\le\theta+1\) for all \(i\). Therefore \(\theta\le x_{(1)}\) and \(\theta\ge x_{(n)}-1\). The likelihood is positive on \([x_{(n)}-1,x_{(1)}]\), if this interval is nonempty.

10. Normal mean with known variance
ISI-styleQuadratic

\(X_1,\ldots,X_n\) are iid \(N(\mu,\sigma^2)\), where \(\sigma^2\) is known. Find the MLE of \(\mu\).

Solution

The log-likelihood differs from a constant by \(-\frac{1}{2\sigma^2}\sum_i(x_i-\mu)^2\). Maximizing it is the same as minimizing the squared deviations, giving \(\widehat\mu=\overline X\).
