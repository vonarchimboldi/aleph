Core Pattern

The maximum likelihood estimator is the parameter value that makes the observed data most likely.

\[ L(\theta)=\prod_{i=1}^n f_\theta(x_i), \qquad \hat\theta=\operatorname*{arg\,max}_\theta L(\theta). \]

For regular problems, maximize the log-likelihood by differentiation. For support-dependent problems, first find which parameter values make the sample possible.

Checklist

  • Write the sample likelihood.
  • Drop factors that do not depend on the parameter.
  • Use the log-likelihood when products are messy.
  • Check the parameter space and support restrictions.
  • For boundaries, compare endpoints and critical points.
  • After estimating, compute bias, variance, or MSE if asked.
Problems 1-5 · Mechanics
1. Bernoulli Likelihood
Mechanics Bernoulli MLE 6 min

Let \(X_1,\ldots,X_n\) be independent Bernoulli\((p)\). If \(\sum_{i=1}^n X_i=s\), find the MLE of \(p\).

Solution

The likelihood is

\[ L(p)=p^s(1-p)^{n-s},\qquad 0The log-likelihood is

\[ \ell(p)=s\log p+(n-s)\log(1-p). \]

Differentiate and set equal to zero:

\[ \ell'(p)=\frac{s}{p}-\frac{n-s}{1-p}=0. \]

This gives \(s(1-p)=(n-s)p\), so \(s=np\). Therefore

\[ \hat p=\frac{s}{n}=\bar X. \]
2. Exponential Rate
Mechanics Log-likelihood 7 min

Let \(X_1,\ldots,X_n\) be independent Exponential\((\lambda)\), with density \(f(x)=\lambda e^{-\lambda x}\) for \(x>0\). Find the MLE of \(\lambda\).

Solution

The likelihood is

\[ L(\lambda)=\lambda^n e^{-\lambda\sum x_i},\qquad \lambda>0. \]

The log-likelihood is

\[ \ell(\lambda)=n\log\lambda-\lambda\sum x_i. \]

Differentiate:

\[ \ell'(\lambda)=\frac{n}{\lambda}-\sum x_i. \]

Set this equal to zero:

\[ \hat\lambda=\frac{n}{\sum x_i}=\frac{1}{\bar X}. \]
3. Normal Mean With Known Variance
Mechanics Normal MLE 8 min

Let \(X_1,\ldots,X_n\) be independent Normal\((\mu,\sigma^2)\), where \(\sigma^2\) is known. Find the MLE of \(\mu\).

Solution

Only the squared-error part of the likelihood depends on \(\mu\). The log-likelihood equals a constant minus

\[ \frac{1}{2\sigma^2}\sum_{i=1}^n (x_i-\mu)^2. \]

So maximizing the likelihood is the same as minimizing

\[ \sum_{i=1}^n (x_i-\mu)^2. \]

Differentiate the sum of squares:

\[ \frac{d}{d\mu}\sum_{i=1}^n(x_i-\mu)^2 = -2\sum_{i=1}^n(x_i-\mu). \]

Set it equal to zero. Then \(\sum x_i=n\mu\), so

\[ \hat\mu=\bar X. \]
4. Uniform Upper Endpoint
Mechanics Support-dependent 8 min

Let \(X_1,\ldots,X_n\) be independent Uniform\((0,\theta)\). Find the MLE of \(\theta\).

Solution

The density of one observation is \(1/\theta\) if \(0

The likelihood is

\[ L(\theta)=\theta^{-n}\mathbf{1}\{\theta\ge x_{(n)}\}. \]

For allowed \(\theta\), the value \(\theta^{-n}\) decreases as \(\theta\) increases. Therefore the largest likelihood occurs at the smallest allowed \(\theta\):

\[ \hat\theta=X_{(n)}. \]
5. Bias and MSE of the Sample Mean
Mechanics Estimator quality 8 min

Let \(X_1,\ldots,X_n\) be independent with mean \(\mu\) and variance \(\sigma^2\). For \(T=\bar X\), find the bias and MSE of \(T\) as an estimator of \(\mu\).

Solution

The bias is \(E[T]-\mu\). Since \(E[\bar X]=\mu\),

\[ \operatorname{Bias}(\bar X)=0. \]

The MSE is

\[ E[(\bar X-\mu)^2]. \]

Because the estimator is unbiased, MSE equals variance:

\[ \operatorname{MSE}(\bar X)=\operatorname{Var}(\bar X)=\frac{\sigma^2}{n}. \]
Problems 6-8 · Applications
6. Poisson Mean
Application Count model 10 min

Let \(X_1,\ldots,X_n\) be independent Poisson\((\lambda)\). Find the MLE of \(\lambda\).

Solution

The likelihood is

\[ L(\lambda)=\prod_{i=1}^n e^{-\lambda}\frac{\lambda^{x_i}}{x_i!}. \]

Drop the factorials because they do not depend on \(\lambda\):

\[ L(\lambda)\propto e^{-n\lambda}\lambda^{\sum x_i}. \]

The log-likelihood, up to constants, is

\[ \ell(\lambda)=-n\lambda+\left(\sum x_i\right)\log\lambda. \]

Differentiate and set equal to zero:

\[ -n+\frac{\sum x_i}{\lambda}=0. \]

So

\[ \hat\lambda=\frac{1}{n}\sum_{i=1}^n X_i=\bar X. \]
7. Constrained Bernoulli Parameter
Application Constraint 12 min

In a Bernoulli sample of size \(n=20\), the number of successes is \(s=14\). The parameter is known to satisfy \(0

Solution

Without the constraint, the Bernoulli MLE is

\[ \hat p_{\text{free}}=\frac{s}{n}=\frac{14}{20}=0.7. \]

But the parameter space only allows \(p\le0.6\). For the Bernoulli likelihood, the maximum over the allowed interval is the allowed value closest to the free optimizer when the free optimizer is outside the interval.

Therefore the constrained MLE is the boundary point

\[ \hat p=0.6. \]
8. Support-Dependent Lower Endpoint
Application Boundary MLE 13 min

Let \(X_1,\ldots,X_n\) be independent Uniform\((\theta,\theta+1)\). Find the set of MLEs for \(\theta\).

Solution

The density is 1 on the interval \((\theta,\theta+1)\). So the likelihood is either 1 or 0.

The sample is possible exactly when every observation lies between \(\theta\) and \(\theta+1\):

\[ \theta\le x_i\le \theta+1 \quad\text{for all }i. \]

This gives two restrictions:

\[ \theta\le x_{(1)} \quad\text{and}\quad \theta\ge x_{(n)}-1. \]

Therefore every value in the interval

\[ x_{(n)}-1\le \theta\le x_{(1)} \]

is an MLE. If the sample range is larger than 1, the model assigns likelihood 0 for all \(\theta\), so the data are impossible under the model.

Problems 9-10 · ISI-Style
9. Sufficiency Cue From Factorization
ISI-style Sufficiency 16 min

Let \(X_1,\ldots,X_n\) be independent Bernoulli\((p)\). Use the factorization idea to identify a sufficient statistic for \(p\).

Solution

The joint likelihood is

\[ L(p)=p^{\sum x_i}(1-p)^{n-\sum x_i}. \]

The data enter the parameter-dependent part only through

\[ T=\sum_{i=1}^n X_i. \]

So, by the factorization idea, \(T=\sum X_i\) is sufficient for \(p\).

The meaning is simple: once we know the total number of successes, the order of successes and failures gives no additional information about \(p\).

10. Rao-Blackwell Recognition
ISI-style UMVUE cue 18 min

In a Bernoulli\((p)\) sample, \(X_1\) is an unbiased estimator of \(p\). Let \(T=\sum_{i=1}^n X_i\). Compute \(E[X_1\mid T]\), and explain why this improves the estimator.

Solution

Given \(T=t\), exactly \(t\) of the \(n\) positions are successes. By symmetry, each position is equally likely to be one of those successes.

\[ P(X_1=1\mid T=t)=\frac{t}{n}. \]

Since \(X_1\) is a 0-1 variable, its conditional expectation is this probability:

\[ E[X_1\mid T=t]=\frac{t}{n}. \]

Therefore

\[ E[X_1\mid T]=\frac{T}{n}=\bar X. \]

This estimator is still unbiased for \(p\), and Rao-Blackwellization cannot increase variance. The improved estimator is \(\bar X\). In the Bernoulli family, this is the standard UMVUE cue.

Answer Summary
  1. \(\hat p=\bar X=s/n\).
  2. \(\hat\lambda=n/\sum X_i=1/\bar X\).
  3. \(\hat\mu=\bar X\).
  4. \(\hat\theta=X_{(n)}\).
  5. Bias \(0\), MSE \(\sigma^2/n\).
  6. \(\hat\lambda=\bar X\) for Poisson\((\lambda)\).
  7. The constrained MLE is \(0.6\).
  8. Every \(\theta\in[X_{(n)}-1,X_{(1)}]\) is an MLE.
  9. \(\sum X_i\) is sufficient for \(p\).
