Core Pattern

Conditional expectation means: compute the expected value after some information is known.

The tower property says that if we first condition on a useful variable \(Y\), and then average over \(Y\), we get the original expectation:

\[ E[X]=E[E[X\mid Y]]. \]

The main skill is choosing \(Y\). A good \(Y\) makes the inside problem easy.

How To Use It

  • Ask: what random thing, if known, makes the problem easy?
  • Call that random thing \(Y\).
  • Compute \(E[X\mid Y]\).
  • Then take expectation again: \(E[E[X\mid Y]]\).
  • For variance, use total variance.
  • Write the conditioning variable before doing arithmetic.
\[ \operatorname{Var}(X) =E[\operatorname{Var}(X\mid Y)] +\operatorname{Var}(E[X\mid Y]). \]
Problems 1-5 · Mechanics
1. Read a Conditional Table
Mechanics Tower property 5 min

A random variable \(Y\) takes values \(1,2,3\) with probabilities \(1/2,1/3,1/6\). Also \(E[X\mid Y=y]=2y+1\). Find \(E[X]\).

Solution

The useful information is already given: condition on \(Y\).

Use the tower property:

\[ E[X]=E[E[X\mid Y]]. \]

Here \(E[X\mid Y]=2Y+1\), so

\[ E[X]=E[2Y+1]=2E[Y]+1. \]

Now compute \(E[Y]\):

\[ E[Y]=1\cdot\frac12+2\cdot\frac13+3\cdot\frac16=\frac12+\frac23+\frac12=\frac53. \]

Therefore

\[ E[X]=2\cdot\frac53+1=\frac{13}{3}. \]
2. Random Number of Coin Tosses
Mechanics Random sum 7 min

A fair coin is tossed \(N\) times, where \(N\) is a nonnegative integer-valued random variable with \(E[N]=10\). Given \(N\), let \(H\) be the number of heads. Find \(E[H]\).

Solution

If we know \(N=n\), the problem becomes easy. We are counting heads in \(n\) fair tosses.

\[ E[H\mid N=n]=\frac n2. \]

So \(E[H\mid N]=N/2\). Now use the tower property:

\[ E[H]=E[E[H\mid N]]=E\left[\frac N2\right]=\frac{E[N]}2=5. \]
3. Roll First, Then Toss
Mechanics Condition on count 8 min

A fair die is rolled. If the die shows \(K\), then a fair coin is tossed \(K\) times. Let \(H\) be the number of heads. Find \(E[H]\).

Solution

Condition on the die result \(K\). If \(K=k\), then the expected number of heads is \(k/2\).

\[ E[H\mid K=k]=\frac k2. \]

So \(E[H\mid K]=K/2\). Use the tower property:

\[ E[H]=E\left[\frac K2\right]=\frac{E[K]}2. \]

For a fair die, \(E[K]=7/2\). Therefore

\[ E[H]=\frac{7}{4}. \]
4. Total Variance With a Random Count
Mechanics Total variance 10 min

In Problem 2, assume also \(\operatorname{Var}(N)=4\). Find \(\operatorname{Var}(H)\).

Solution

Again condition on \(N\). If \(N=n\), then \(H\mid N=n\) is binomial with \(n\) tosses and success probability \(1/2\).

\[ E[H\mid N]=\frac N2,\qquad \operatorname{Var}(H\mid N)=\frac N4. \]

Use total variance:

\[ \operatorname{Var}(H) =E[\operatorname{Var}(H\mid N)] +\operatorname{Var}(E[H\mid N]). \]

Substitute the two conditional quantities:

\[ \operatorname{Var}(H)=E\left[\frac N4\right]+\operatorname{Var}\left(\frac N2\right) =\frac{E[N]}4+\frac{\operatorname{Var}(N)}4. \]

With \(E[N]=10\) and \(\operatorname{Var}(N)=4\),

\[ \operatorname{Var}(H)=\frac{10}{4}+\frac44=\frac72. \]
5. Conditional Distribution
Mechanics Discrete conditioning 10 min

A random variable \(Y\) takes values \(0\) and \(1\) with \(P(Y=1)=p\). Given \(Y\), the random variable \(X\) has \(E[X\mid Y=0]=3\) and \(E[X\mid Y=1]=8\). Find \(E[X]\).

Solution

We average the conditional expectations using the probabilities of the conditioning cases.

\[ E[X]=E[E[X\mid Y]]. \]

There are two cases:

\[ E[X]=(1-p)\cdot3+p\cdot8. \]

So

\[ E[X]=3+5p. \]
Problems 6-8 · Applications
6. Random Box Choice
Application Mixture 14 min

There are two boxes. Box A has 2 red and 3 blue balls. Box B has 5 red and 5 blue balls. A box is chosen at random, each with probability \(1/2\). Then 4 balls are drawn with replacement from that box. Let \(R\) be the number of red balls drawn. Find \(E[R]\).

Solution

The box choice changes the red probability. So condition on the chosen box.

If Box A is chosen, the red probability is \(2/5\), so

\[ E[R\mid A]=4\cdot\frac25=\frac85. \]

If Box B is chosen, the red probability is \(1/2\), so

\[ E[R\mid B]=4\cdot\frac12=2. \]

Now average over the box choice:

\[ E[R]=\frac12\cdot\frac85+\frac12\cdot2=\frac45+1=\frac95. \]
7. Random Defect Rate
Application Random parameter 15 min

A machine first chooses a defect probability \(P\). The random value \(P\) is uniformly distributed on \([0,1]\). Given \(P=p\), the machine makes 12 items independently, each defective with probability \(p\). Let \(D\) be the number of defective items. Find \(E[D]\) and \(\operatorname{Var}(D)\).

Solution

Condition on \(P\). Once \(P=p\) is known, \(D\mid P=p\) is binomial with 12 trials and success probability \(p\).

\[ E[D\mid P]=12P,\qquad \operatorname{Var}(D\mid P)=12P(1-P). \]

First find the expectation:

\[ E[D]=E[12P]=12E[P]=12\cdot\frac12=6. \]

For variance, use total variance:

\[ \operatorname{Var}(D)=E[12P(1-P)]+\operatorname{Var}(12P). \]

For \(P\sim \operatorname{Unif}(0,1)\), \(E[P]=1/2\), \(E[P^2]=1/3\), and \(\operatorname{Var}(P)=1/12\). Therefore

\[ E[12P(1-P)]=12\left(E[P]-E[P^2]\right)=12\left(\frac12-\frac13\right)=2. \]

Also

\[ \operatorname{Var}(12P)=144\cdot\frac1{12}=12. \]

So

\[ \operatorname{Var}(D)=2+12=14. \]
8. Waiting for a Random Target
Application Geometric 16 min

A fair die is rolled once. Suppose the result is \(K\). Then the die is rolled repeatedly until \(K\) appears again. Let \(T\) be the number of repeated rolls needed. Find \(E[T]\).

Solution

Condition on the first roll \(K\). Once \(K\) is known, each later roll has probability \(1/6\) of matching \(K\).

So \(T\mid K\) is geometric with success probability \(1/6\). Its mean is

\[ E[T\mid K]=6. \]

This does not depend on the value of \(K\). Now average:

\[ E[T]=E[E[T\mid K]]=E[6]=6. \]
Problems 9-10 · Hidden Tower Property
9. Product of a Random Sum
Hidden pattern Conditioning choice 22 min

Let \(N\) be a nonnegative integer-valued random variable with \(E[N]=\mu\) and \(E[N^2]<\infty\). Let \(X_1,X_2,\ldots\) be independent random variables, independent of \(N\), with \(E[X_i]=m\). Define \(S=X_1+\cdots+X_N\), with \(S=0\) when \(N=0\). Find \(E[NS]\).

Solution

The expression \(NS\) looks like a product, so the tower property is not obvious at first. The useful variable to freeze is \(N\).

If \(N=n\), then

\[ S=X_1+\cdots+X_n. \]

So

\[ E[S\mid N=n]=nm. \]

Now compute the conditional expectation of \(NS\):

\[ E[NS\mid N=n]=nE[S\mid N=n]=n^2m. \]

Therefore \(E[NS\mid N]=mN^2\). Average over \(N\):

\[ E[NS]=E[mN^2]=mE[N^2]. \]
10. Random Cut in a Stick
Hidden pattern Condition on geometry 25 min

A point \(U\) is chosen uniformly on \([0,1]\). Then, conditional on \(U=u\), a point \(V\) is chosen uniformly on \([0,u]\). Find \(E[V]\) and \(\operatorname{Var}(V)\).

Solution

The variable \(V\) is hard to understand directly because its range depends on \(U\). So condition on \(U\).

If \(U=u\), then \(V\) is uniform on \([0,u]\). Therefore

\[ E[V\mid U=u]=\frac u2,\qquad \operatorname{Var}(V\mid U=u)=\frac{u^2}{12}. \]

So

\[ E[V\mid U]=\frac U2,\qquad \operatorname{Var}(V\mid U)=\frac{U^2}{12}. \]

First find the mean:

\[ E[V]=E\left[\frac U2\right]=\frac12E[U]=\frac14. \]

Now use total variance:

\[ \operatorname{Var}(V)=E\left[\frac{U^2}{12}\right]+\operatorname{Var}\left(\frac U2\right). \]

For \(U\sim \operatorname{Unif}(0,1)\), \(E[U^2]=1/3\) and \(\operatorname{Var}(U)=1/12\). Thus

\[ \operatorname{Var}(V)=\frac1{12}\cdot\frac13+\frac14\cdot\frac1{12} =\frac1{36}+\frac1{48} =\frac7{144}. \]
