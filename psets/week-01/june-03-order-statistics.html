Core Pattern

Start with independent samples \(X_1,\ldots,X_n\). Sort them:

\[ X_{(1)} \le X_{(2)} \le \cdots \le X_{(n)}. \]

The smallest value is \(X_{(1)}\). The largest value is \(X_{(n)}\). The kth order statistic is \(X_{(k)}\).

For many problems, do not try to write the density first. Translate the event into a statement about how many original observations fall below a cutoff.

Recognition Guide

  • For the maximum, use \(P(X_{(n)}\le x)=P(X_1\le x,\ldots,X_n\le x)\).
  • For the minimum, use \(P(X_{(1)}>x)=P(X_1>x,\ldots,X_n>x)\).
  • For the kth value, count how many observations are at most \(x\).
  • For Uniform\((0,1)\), \(X_{(k)}\) has a beta shape.
  • For exponential samples, spacings between sorted values have useful exponential laws.

1. Maximum by the CDF Method
Concept Maximum 6 min

2. Minimum by the Survival Method
Concept Minimum 6 min

Let \(X_1,\ldots,X_4\) be independent Uniform\((0,1)\). Let \(L=X_{(1)}\). Find \(P(L>t)\) and \(P(L\le t)\) for \(0\le t\le1\).

Solution

The event \(L>t\) means the smallest sample value is above \(t\). That can happen only if all four observations are above \(t\).

\[ P(L>t)=P(X_1>t,\ldots,X_4>t). \]

Each observation is above \(t\) with probability \(1-t\). By independence,

\[ P(L>t)=(1-t)^4. \]

Therefore the CDF is

\[ P(L\le t)=1-(1-t)^4. \]
3. Median of Three Uniforms
Concept kth order statistic 8 min

Let \(X_1,X_2,X_3\) be independent Uniform\((0,1)\). Let \(Y=X_{(2)}\), the middle value. Find \(P(Y\le y)\) for \(0\le y\le1\).

Solution

The middle value is at most \(y\) when at least two observations are at most \(y\).

The count of observations at most \(y\) is Binomial\((3,y)\). So

\[ P(Y\le y)=P(\operatorname{Binomial}(3,y)\ge2). \]

There are two cases: exactly two are at most \(y\), or all three are at most \(y\).

\[ P(Y\le y)=\binom32y^2(1-y)+y^3=3y^2(1-y)+y^3. \]

Simplify:

\[ P(Y\le y)=3y^2-2y^3. \]
4. Density of the kth Uniform Order Statistic
Concept Beta link 9 min

Let \(U_{(k)}\) be the kth order statistic from \(n\) independent Uniform\((0,1)\) samples. Write the density of \(U_{(k)}\).

Solution

For \(U_{(k)}\) to be near \(u\), we need \(k-1\) observations below \(u\), one observation near \(u\), and \(n-k\) observations above \(u\).

The density is

\[ f_{U_{(k)}}(u) = \frac{n!}{(k-1)!(n-k)!} u^{k-1}(1-u)^{n-k}, \qquad 0This is the Beta\((k,n-k+1)\) density.

5. Expected Maximum of Uniform Samples
Concept Expectation 8 min

Let \(M=X_{(n)}\) be the maximum of \(n\) independent Uniform\((0,1)\) observations. Find \(E[M]\).

Solution

From the maximum CDF method, \(F_M(m)=m^n\) on \(0\le m\le1\). The density is

\[ f_M(m)=nm^{n-1}. \]

Now compute the expectation:

\[ E[M]=\int_0^1 m\cdot nm^{n-1}\,dm =n\int_0^1m^n\,dm =\frac{n}{n+1}. \]

7. Exponential Minimum
Application Exponential race 12 min

Let \(X_1,\ldots,X_n\) be independent Exponential\((\lambda)\) random variables. Let \(L=X_{(1)}\). Find \(P(L>t)\), identify the distribution of \(L\), and find \(E[L]\).

Solution

The minimum is greater than \(t\) only if every observation is greater than \(t\).

\[ P(L>t)=P(X_1>t,\ldots,X_n>t). \]

For one exponential variable, \(P(X_i>t)=e^{-\lambda t}\). By independence,

\[ P(L>t)=e^{-n\lambda t}. \]

This is the survival function of an Exponential\((n\lambda)\) random variable. Therefore

\[ L\sim \operatorname{Exponential}(n\lambda), \qquad E[L]=\frac{1}{n\lambda}. \]

Problems 9-10 · Challenge
9. Hidden Minimum in a Coverage Problem
Challenge Hidden order statistic 18 min

Choose \(n\) points independently and uniformly from \([0,1]\). Let \(L\) be the smallest chosen point. Find the expected length of the interval from 0 to the first chosen point.

Solution

The interval from 0 to the first chosen point has length \(L=X_{(1)}\), the minimum order statistic.

For the minimum, use the survival function:

\[ P(L>t)=(1-t)^n,\qquad 0\le t\le1. \]

For a nonnegative variable on \([0,1]\),

\[ E[L]=\int_0^1 P(L>t)\,dt. \]

So

\[ E[L]=\int_0^1(1-t)^n\,dt=\frac{1}{n+1}. \]

The expected first gap is \(1/(n+1)\).

10. ISI-Style Range of Uniform Points
ISI-style Range 22 min

Let \(X_1,\ldots,X_n\) be independent Uniform\((0,1)\). Let \(R=X_{(n)}-X_{(1)}\) be the sample range. Find \(P(R\le r)\) for \(0\le r\le1\).

Solution

Use the joint density of the minimum \(U=X_{(1)}\) and maximum \(V=X_{(n)}\):

\[ f_{U,V}(u,v)=n(n-1)(v-u)^{n-2}, \qquad 0The event \(R\le r\) means \(v-u\le r\).

It is easier to integrate over the range length \(s=v-u\). For a fixed \(s\), the left endpoint \(u\) can run from \(0\) to \(1-s\). Therefore

\[ P(R\le r)= \int_0^r\int_0^{1-s} n(n-1)s^{n-2}\,du\,ds. \]

The inner integral contributes \(1-s\):

\[ P(R\le r)=n(n-1)\int_0^r s^{n-2}(1-s)\,ds. \]

Compute the integral:

\[ P(R\le r) = n(n-1)\left(\frac{r^{n-1}}{n-1}-\frac{r^n}{n}\right) = nr^{n-1}-(n-1)r^n. \]
